AZN.tools
Uptime | Data Source

AZN exchange rates API

in JSON format

Rates are updated regularly from Central Bank of Azerbaijan.


Usage

To get latest rates use method below:
GET  http://azn.tools/latest

Result will be in JSON format
{
    "date": "01.01.2020",
    "rates": {
        "commodities": [
            {
                "name": "XAU",
                "value": 
            }
            ...
        ],
        "currencies": [
            {
                "name": "USD",
                "facevalue": 1,
                "value": 
            },
            {
                "name": "EUR",
                "facevalue": 1,
                "value": 
            }
            ...
        ]
    }
}
